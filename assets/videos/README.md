# Videos

Place your project video files here.

## Format
- Use **MP4 (H.264)** for maximum browser compatibility
- For hero/background videos: compress to under **5MB** (use HandBrake or Adobe Media Encoder)
- For project reels: under **30MB**

## Naming convention
video-[name]-[type].mp4

Examples:
- video-hero-background.mp4
- video-reel-2024.mp4
- video-motion-project-name.mp4

## How to use in index.html
```html
<video autoplay loop muted playsinline>
  <source src="assets/videos/video-hero-background.mp4" type="video/mp4">
</video>
```

## Free compression tools
- HandBrake: https://handbrake.fr (desktop, very powerful)
- Clideo: https://clideo.com/compress-video (browser-based)
